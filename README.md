## Description

[Node](https://nodejs.org/en) project to serve json files as endpoints using `json-server`

## Installation

```
npm install
```

## Running the server

```
npm run server
```

## Structure

```
├── src
│   ├── index.js
│   ├── amiibo.json
│   └── ...
├── pnpm-lock.json
├── package.json
└── README.md
```

### Dependencies

- [json-server](https://github.com/typicode/json-server/)
